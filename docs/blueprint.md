# **App Name**: TeamStation AI

## Core Features:

- Content Summarization: Generates concise summaries of playbook content, extracting key insights for quick consumption.
- Personalized Recommendations: Suggests relevant playbook articles based on the user's current role, industry, and specific challenges, using a tool.
- Playbook Article Display: Presents the playbook articles in a clear, readable format with consistent styling and layout.
- Search Functionality: Allows users to search for specific topics or keywords within the playbook content.
- Themed Display: Uses CSS vars from tailwind config to style the site for light and dark modes
- Category Browsing: Enable users to browse related articles by navigating hub pages.

## Style Guidelines:

- Primary color: Dark blue for the site's background.
- Highlight color: Light blue for highlighting key elements.
- Header text: Light blue gradient for a visually appealing header.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, conveying a blend of techy precision and readability. Poppins font should be used throughout the site.
- Code font: 'Source Code Pro' for displaying code snippets.
- The hero headers (H1) should be closer in distance to the content directly underneath.
- Use minimalist icons from 'lucide-react' library to represent different sections and actions, maintaining a clean and modern look.
- Subtle transitions and animations for interactive elements to provide visual feedback and enhance user experience.
